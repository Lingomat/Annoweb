AikumaNG is the result if an international research project to develop tools to help communities document, maintain and revive their 'treasure' languages.

This software is the evolution of Aikuma, an Android mobile-phone app described in detail at the Aikuma Project. The Aikuma name Aikuma means "meeting place" in Usarufa, the language where this software was first used. Usarufa is spoken in the Eastern Highlands Province of Papua New Guinea.

This software uses web technologies exclusively. This beta version offers the modest capability to respeak, translate and annotate speech but there is must more to do. If you are interested in helping, you can do so in a number of ways. We welcome high quality feedback such as reporting of issues and suggesting improve features. Additionally we would be greatly appreciative of assistance in development. Fork AikumaNG on GitHub and contribute back PRs!

AikumaNG would not be possible without the wealth of high quality open source projects upon which it depends:
