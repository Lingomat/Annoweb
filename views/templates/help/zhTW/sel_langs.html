The language selector lets you specify one or more languages in the source recording. To add, start typing the language name and select the language you want from the autocomplete box and a language 'chip' will appear in the language selector.

If your language isn't yet in ISO 639-3 (you don't see anything when you type the name) then just hit return and it will be added manually.

If you want to remove a language, just click the of the language chip you wish to remove.
