New Recording page

Purpose: Users see this page when they want to record a new file. However the behavior is context sensitive depending on whether this is a brand new primary recording, or if it is a new file attached to a previous recording.

Design influence: Any recording app, Aikuma metadata wizard.

Navigation: They get here from the home/landing page by selecting new. Or from the landing page for their primary recording. Recording is a wizard that collects metadata and then returns to the project status page (or creates a new project status page).
